# 🔐 SecurePass

SecurePass is an educational Password Strength Analyzer and Attack Simulator built using HTML, CSS, and JavaScript.

The application evaluates password strength in real time and demonstrates how attackers exploit weak passwords through dictionary attacks, brute-force attacks, and credential stuffing.

## Features

* Real-time password strength analysis
* Password score out of 100
* Animated strength meter
* Password rule checklist
* Password recommendations
* Dictionary attack simulation
* Brute-force attack estimation
* Credential stuffing awareness
* Dark mode support
* Responsive design
* Client-side password analysis

## Cybersecurity Concepts Demonstrated

* Password complexity
* Password entropy
* Common password detection
* Dictionary attacks
* Brute-force attacks
* Credential stuffing
* Secure-by-design principles
* Client-side processing of sensitive information

## Technology Stack

* HTML5
* CSS3
* JavaScript

## Security Notice

Passwords are analyzed entirely within the browser and are never stored, logged, or transmitted.
